# Data_Science_Salaries_dashboard# 

An interactive dashboard built using Streamlit to explore and analyze data science salaries across different experience levels and years.

---

## 🚀 Features
- Filter data by *Experience Level* and *Work Year*
- Display key metrics:
  - 💰 Average Salary
  - 📈 Maximum Salary
  - 📉 Minimum Salary
  - 🧮 Total Salary
- Interactive visualization using Plotly
- Clean and simple user interface

---

## 🛠️ Tech Stack
- Python
- Pandas
- Streamlit
- Plotly

---

## 📂 Dataset
The dataset contains information about data science salaries including:
- Job Title
- Experience Level
- Work Year
- Salary in USD
